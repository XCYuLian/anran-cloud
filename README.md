# 安燃云 - B端Web综合管理中枢

大创赛路演用的液化石油气（LPG）安全管理后台原型。纯前端单文件，双击就能跑。

## 怎么用

```
双击 index.html → 浏览器打开
```

没了。不用 npm install，不用构建，所有依赖走 CDN（Vue3、Tailwind、ECharts、FontAwesome）。

断网的话首次加载会挂，但只要你开过一次浏览器缓存住了就能离线跑。

## 技术栈

| 层 | 方案 |
|---|---|
| 框架 | Vue 3 全局构建（Composition API） |
| 样式 | Tailwind CDN + 自定义玻璃态白底卡片 |
| 图表 | ECharts 5.5 |
| 图标 | FontAwesome 6.5.1 |

## 页面结构

左侧 7 组菜单，点击切换，不走路由，就是 v-if 硬切。

**完整实现的页面（10 个）：**

- **工作台** — 默认首页，5 个 KPI 卡片 + 近 7 日折线柱状混合图（横轴动态取当天日期，第 4 天有峰值标记线）。气瓶总量和新增用户每隔几秒会微跳动，模拟物联网推流
- **政府监管大屏** — 三栏驾驶舱：左侧高危台账滚动列表、中间 ECharts 散点气泡图模拟城市网格监测点、右侧企业合规红黑榜（带进度条）
- **一瓶一码溯源** — 钢瓶列表，点"查看存证"展开区块链存证档案（区块高度、交易哈希、智能合约地址全写死）
- **智能充装记录** — KPI 卡 + 防作弊动态（绿色是通过，红色是电磁阀锁死拦截）
- **智能角阀在线监测** — 状态卡片 + 设备列表
- **报警器漏气动态** — 监测点统计 + 报警列表（含一条高危行模拟真实告警）
- **隐患整改工单** — 状态筛选按钮 + 工单表格
- **入户安检记录** — 左右对比组件：纯 CSS 画的灶台+钢瓶+软管场景，右侧叠加红框模拟计算机视觉分析引擎标注
- **实时订单大厅** — 待派单/配送中/已送达看板 + LBS 路径优化卡片
- **营收数据报表** — SaaS 订阅 + 保险佣金 + 财务明细

**骨架屏占位（8 个）：** 钢瓶电子档案、超期/报废瓶库、LBS自动派单、客户信息档案、亲情账号列表、充装与销售对账、员工与权限管理、气站基础配置

## 项目约束

几个坑，改代码前先看：

1. **别写"AI"两个字。** 评委看到 AI 会追着问数据集、算力、网络架构。统一叫"计算机视觉分析引擎"或"自动化风险决策引擎"
2. **别引外部图片。** Unsplash 在国内加载不稳定。CV 对比场景是纯 CSS 画的（渐变 + border + 阴影），不用任何 url()
3. **别加 npm 包。** 单文件就是单文件，CDN 够用了
4. **菜单里"智能"二字只保留 IoT 硬件层面的。** 比如"智能角阀""智能充装"可以留，但"LBS智能派单"已经改成"LBS自动派单"了

## 已知问题

- 数据全是写死的，没有接后端
- 表格 CSS class 是后加的统一样式，个别单元格有特殊颜色的是单独覆盖的
- ECharts 图表在窗口缩放时会跟着变，但政府大屏那张图切走再切回来要重新初始化
- Firefox 滚动条会比 Chrome 粗一点，不影响使用

## 注意

想改数据的话直接搜对应变量名，全在 `<script>` 里的 setup() 附近。KPI 数字是写死的 const，图表数据也是 const 数组，搜 `gasData = [` 就能找到。

Logo 在 `Pic/纯图标LOGO.png`，想换直接替换文件就行，不用改代码。
