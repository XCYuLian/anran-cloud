# 安燃云 — B端Web综合管理中枢

大创赛路演用液化石油气（LPG）安全管理后台原型。
纯前端单文件演示系统，双击 `index.html` 即可在浏览器运行。

## 技术栈

| 层 | 方案 |
|---|---|
| 架构 | 单文件 HTML5（无构建工具，无打包） |
| UI 框架 | Vue 3 全局构建（Composition API） |
| 样式 | Tailwind CSS CDN + 自定义玻璃态暗夜主题 |
| 图表 | ECharts CDN |
| 图标 | FontAwesome 6.5.1 CSS |

## 运行方式

```
双击 index.html → 浏览器打开即用
```

无 `npm install`，无构建步骤。所有依赖通过 CDN 加载。

## 视觉规范

- **底色**：`#090D16` 极深蓝灰，右上/左下角微弱科技光晕（`#0284C7` + `#22C55E`，透明度 0.03）
- **卡片**：玻璃态（Glassmorphism）— 半透明微渐变 + `backdrop-filter: blur(16px)` + `border: 1px solid rgba(255,255,255,0.08)`
- **图表**：ECharts 荧光霓虹线条，`shadowBlur: 10`，渐变 AreaStyle
- **无纯色色块**，无传统教务系统风格

## 7 大菜单模块

1. 数据中心 — 工作台（默认页）、政府监管大屏接口
2. 钢瓶资产管控 — 电子档案、一瓶一码溯源、智能充装记录、超期/报废瓶库
3. 物联安全预警 — 角阀在线监测、报警器漏气动态、隐患整改工单
4. 订单与调度 — 实时订单大厅、LBS自动派单
5. 客户与安检 — 客户信息档案、入户安检记录、亲情账号列表
6. 财务与结算 — 充装与销售对账、营收数据报表
7. 系统设置 — 员工与权限管理、气站基础配置

**核心完整实现**：工作台、一瓶一码溯源、物联安全预警三页、入户安检记录（6 页）
**其余骨架屏**：其余 12 页用 Skeleton Screen 占位

## 代码铁律

### 关键词禁止
- **严禁出现**：`AI`、`AI视觉`、`AI识别`、`百度EasyDL`、`大模型`、`算法模型`、`神经网络`、`深度学习`
- **替换词**：`计算机视觉分析引擎`、`自动化风险决策引擎`
- **保留**：`智能角阀`、`智能充装` 等 IoT 硬件层面的"智能"可保留
- 菜单中 `LBS智能派单` → `LBS自动派单`

### 交互机制
- 实时数据抖动：气瓶总量(148) ±1 每 4s，新增用户(874) ±3~5 每 3s
- ECharts 横轴用 `new Date()` 动态生成近 7 天，非写死日期
- 第 4 天做 markLine 高亮 + 悬浮框显示气瓶量 178

### 预警彩蛋
- Header 铃铛图标（`fa-bell`）双击触发
- 全局红色微光闪烁 + 中央模态框弹出
- 显示报警信息 + 两条自动完成进度条
- 点击"确认知悉"后淡出恢复

## 关键组件位置

- `入户安检记录` 页：计算机视觉隐患判定对比组件（左原图 / 右 CSS 红框标注）
- `一瓶一码溯源轨迹` 页：区块链存证区块（蚂蚁链/长安链，含区块高度+TxHash）
- `工作台` 页：KPI 卡片 + ECharts 混合图表 + 实时数据
